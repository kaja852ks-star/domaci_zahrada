# Domácí zahrada

Domácí zahrada je jednoduchá webová stránka zaměřená na pěstování rostlin a bylinek v domácím prostředí.

Web je vytvořen jako statický projekt pomocí HTML, CSS a JavaScriptu a je publikován prostřednictvím GitHub Pages.

## Obsah webu

Na stránce se nachází několik hlavních sekcí:

Tipy pro rostliny: základní rady a doporučení pro péči o pokojové rostliny  
Bylinky: informace o pěstování bylinek v domácích podmínkách  
Návody: jednoduché postupy a praktické tipy krok za krokem  
Kontakt: možnost spojení s autorem nebo správcem webu  

## Použité technologie

HTML: struktura a obsah webové stránky  
CSS: stylování, barvy a rozložení prvků  
JavaScript: základní interaktivita stránky  
GitHub Pages: nasazení a zveřejnění projektu  

## Cíl projektu

Cílem projektu je vytvořit přehlednou webovou stránku, která slouží jako inspirace pro domácí zahradničení a může být využita jako školní nebo studijní projekt.
